Data files
"These data files are made available in two formats:
i) a single MS Excel workbook with multiple tabs; or 
ii) a zip file containing a set of comma-separated (csv) data files.
In either format, the user will receive a readme document, a data dictionary document, a codeset document, a data file containing 2008 data and a data file containing 2010 data.  Data for both 2008 and 2010 were compiled afresh for the WHO Report on the Global Tobacco Epidemic, 2011. "

Data Dictionary
A data dictionary file in text format is provided.  It contains descriptors for field names in both the 2008 data file and the 2010 data file.  It also notes the data format for each field and whether the field is a raw response or a calculated field.  Please note that some of the field names have changed since the last report.

Code Sets
A codeset file in text format is provided.  It contains descriptors for field values in both the 2008 data file and the 2010 data file. Fields where the values are self-descriptive are excluded from this file.

End of file.
